# Will Fintech

Project of a fictional fintech with the purpose of self-study

## Resources

- Conta Corrente
  - Saldo
  - Transferências
  - Depósitos
- Cartão de crédito
  - Simulação de compra
  - Parcelamento
  - Geração de faturas

## Running the app

```bash
# development
$ yarn run start

# accounts app in dev mode
$ yarn run start:accounts:dev

# cashflow app in dev mode
$ yarn run start:cashflow:dev
```
